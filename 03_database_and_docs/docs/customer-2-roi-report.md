# CUSTOMER #2 ROI REPORT

## Objective

Customer #2에게 Zenov 도입의 경제적 가치를 설명한다.

## ROI Categories

### Operational Efficiency

- 수작업 보고 시간 절감
- CSV 정리 시간 절감
- 운행 데이터 검증 자동화

### MRV Automation

- Evidence 자동 생성
- MRV 자동 계산
- Report 자동 생성

### Carbon Asset Candidate

- 검증 가능한 감축 실적 생성
- Asset Candidate 수량 산정
- 예상 Carbon Value 산정

### Management Visibility

- 차량별 운행 데이터 확인
- 기사별 데이터 확인
- 실패 Row와 오류 원인 확인

## ROI Formula

```text
Total Value
= Manual Work Reduction
+ Reporting Cost Reduction
+ MRV Automation Value
+ Estimated Carbon Asset Value
```

## Required Inputs

- 차량 수
- 월 운행일수
- 일일 평균 주행거리
- 수작업 보고 시간
- ESG/MRV 보고 비용
- 전기차 전환 계획

## Executive Message

Zenov는 고객에게 소프트웨어 화면을 파는 것이 아니라, 운영 데이터를 검증 가능한 탄소 자산 후보와 자동 보고서로 전환하는 경제적 가치를 제공한다.
