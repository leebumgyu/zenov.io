# CUSTOMER #2 CASE STUDY

## One-Line Story

Customer #2는 Customer Zero에서 검증된 Zenov Trust Chain을 30일 안에 복제하여 첫 Carbon Asset Candidate와 MRV Report를 생성한다.

## Before

- 운행 데이터가 분산되어 있음
- 수작업 검증 필요
- MRV 산정 근거 부족
- 탄소 자산 후보 관리 체계 없음

## After

```text
Data
-> Evidence
-> MRV
-> Verification
-> Asset Candidate
-> Report
```

## Measured Outcomes

| Metric | Target |
| --- | --- |
| Onboarding Time | < 7 Days |
| First Evidence | < 24 Hours |
| First MRV | < 7 Days |
| First Asset | < 30 Days |
| Traceability | 100% |

## Proof Points

- 실제 고객 데이터 사용
- 실패 Row reason_code 보존
- Manual Audit 가능
- Report Snapshot 생성

## Closing Message

Customer #2는 Zenov가 단일 프로젝트가 아니라 반복 가능한 Carbon Data Network Platform임을 증명하는 첫 복제 사례다.
