MODULE 01

탄소중립과
기업의 관계

📌 왜 지금 기업이 움직여야 하나

탄소 문제는 더 이상 환경부서의 일이 아닙니다. 글로벌 규제·고객 요구·투자 조건이 동시에 바뀌고 있습니다.

⚡ 지금 준비하지 않으면 발생하는 리스크
EU CBAM(탄소국경조정제도): 철강·시멘트·알루미늄 등 수출 시 탄소 비용 부과
글로벌 대기업 공급망: Scope 3 관리 위해 협력사 탄소 데이터 의무 제출 요구
ESG 평가 하락: 금융기관 대출·투자 조건에 ESG 등급 반영
공공입찰 불이익: 정부·공기업 발주 시 ESG 인증 가점 확대
💡 탄소 데이터가 없으면 생기는 일
탄소 데이터 없음
→
Scope 3 보고 불가
→
공급망 탈락
배출량 미검증
→
ESG 평가 하락
→
금융 비용 증가
핵심: 탄소 데이터는 이제 재무 데이터처럼 관리해야 하는 경영 자산입니다.

MODULE 02

ESG란
무엇인가

🌐 ESG 3대 축
구분의미탄소와 연결
E 환경기후변화·탄소·에너지·폐기물직접 연결 — 배출량 관리
S 사회노동·인권·지역사회·공급망간접 연결 — 공급망 기준
G 지배구조이사회·투명성·반부패데이터 신뢰성·공시 체계
실무 핵심: 투자자와 고객은 E(환경) 지표, 특히 탄소 배출량을 가장 먼저 봅니다.
📋 주요 ESG 공시 기준
기준특징적용 대상
GRI가장 광범위한 지속가능성 보고전 세계 기업
TCFD기후 관련 재무 정보 공시금융·대형 기업
CSRDEU 지속가능성 보고 지침EU 사업 기업
K-ESG국내 ESG 평가 가이드라인국내 기업

MODULE 03

Scope 1, 2, 3
배출량 이해

📊 Scope 분류 체계
SCOPE 1
직접 배출
회사가 직접 소유·통제하는 시설
자체 보일러, 사내 차량
SCOPE 2
간접 배출
구매한 전기·열·증기 생산
전기 사용량
SCOPE 3
가치사슬 배출
공급망 상·하류 전체
협력사, 물류, 고객 사용
Scope 3이 핵심: 대부분 기업의 전체 탄소 발자국 중 70~90%가 Scope 3에 해당합니다. 공급망 협력사 데이터 없이는 관리 불가능합니다.
🧮 배출량 계산 기본 공식
배출량(tCO₂e) = 활동 데이터 × 배출계수
항목활동 데이터배출계수
전기 사용kWh국가 전력 배출계수
경유 연소리터(L)2.6 kgCO₂/L
LNG 사용Nm³2.2 kgCO₂/Nm³

MODULE 04

MRV: 측정·보고·검증
데이터 신뢰성

🔍 MRV란 무엇인가
M
Measurement
실제 배출량을 정확하게 측정
R
Reporting
국제 기준에 맞게 보고
V
Verification
제3자가 독립적으로 검증
MRV가 없으면: 감축을 해도 인정받지 못하고, 크레딧을 발행할 수 없으며, 보고서의 신뢰성이 없습니다.
⚠️ 그린워싱(Greenwashing) 리스크

검증되지 않은 탄소 주장은 그린워싱으로 제재받을 수 있습니다.

그린워싱으로 판단되는 사례
"탄소중립 제품"이라 표시했으나 근거 데이터 없음
Scope 1,2만 줄이고 Scope 3는 무시한 채 "넷제로 달성" 주장
검증받지 않은 자체 추정치로 감축 효과 홍보

MODULE 05

공급망
탄소 관리

🔗 고객이 탄소 데이터를 요구하는 이유

삼성·현대·LG·애플·BMW 등 글로벌 대기업들은 공급망 전체의 Scope 3 배출량을 보고해야 합니다. 협력사 데이터 없이는 보고 불가능합니다.

현실: "탄소 데이터 제출 못하면 공급망에서 제외합니다"라는 요구가 이미 한국 기업들에 도달하고 있습니다.
📋 고객 요구 대응 체크리스트
✓
연간 온실가스 인벤토리 작성
Scope 1, 2 배출량을 GHG Protocol에 따라 계산
✓
주요 협력사 배출량 파악
상위 80% 구매처의 탄소 데이터 수집
✓
제3자 검증(Verification) 수행
공인된 검증기관을 통한 데이터 신뢰성 확보
✓
CDP(탄소정보공개프로젝트) 응답
글로벌 대기업 공급망 참여의 핵심 요건
✓
감축 목표(SBTi) 설정
과학 기반 감축 목표로 국제 신뢰성 확보

MODULE 06

지금 당장
해야 할 것

🚀 실무 우선순위 3단계
1단계: 측정
→
Scope 1·2 배출량 계산부터
2단계: 체계화
→
데이터 관리 시스템 구축 + 검증
3단계: 감축
→
감축 목표 설정 + 공급망 확대
가장 먼저 해야 할 일: 현재 우리 회사의 Scope 1·2 배출량이 얼마인지 파악하는 것입니다. 측정 없이는 감축도, 보고도, 인증도 없습니다.
📌 즉시 확인해야 할 항목
✓
전기·가스·연료 사용량 데이터 수집
최소 최근 3년치 에너지 사용 청구서 확보
✓
담당자 지정 및 내부 교육
ESG/탄소관리 담당 인력 지정
✓
주요 고객사의 탄소 요구사항 파악
CDP, EcoVadis 등 고객 요청 기준 확인
✓
배출권거래제 대상 여부 확인
연간 5만tCO₂ 이상 사업장은 의무 대상
