# ZENOV Vercel Frontend

Upload the contents of this folder to the Vercel frontend repository root.

This folder is already fixed for Vercel root deployment:

- CSS uses `./app.css`
- JavaScript uses `./app.js`
- App links do not use `/demo-web`
- API calls can be proxied to `https://api.zenov.io`

Expected GitHub/Vercel root:

```text
index.html
app.css
app.js
partner/
education/
vercel.json
```
